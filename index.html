OntoHate Leve • Pesado • Inteligente

Este projeto representa o discurso de ódio online por meio da ontologia OntoHate, permitindo: (i) gerar taxonomias por dimensão e por cruzamento (nível leve), (ii) explorar a ontologia como grafo interativo com relações/propriedades (nível pesado), (iii) responder questões de competência via SPARQL (nível inteligente).

Controles — Taxonomia

Como usar (nível leve)
  • Escolha uma ou mais dimensões e gere a taxonomia.
  • Use cruzamento para mostrar apenas classes que pertencem às dimensões selecionadas.
  • Arraste os nós para organizar. Escolha layout Top-down, Radial ou Force.
  • Edite cores por dimensão e exporte a visualização (PNG/SVG).
Dica: segure Ctrl/Cmd para selecionar múltiplas.

Controles — Ontologia

Como usar (nível pesado)
  • Selecione uma dimensão para focar (ou todas).
  • Ative/Desative subClassOf e Object Properties (arestas com rótulos).
  • Passe o mouse em uma classe para ver todas as anotações (tooltip).
  • Clique para fixar e ver detalhes no painel.
